# RED 3.3 guide

> Stub. See the concept plan ("RED 3.3 (Article 3.3(d)(e)(f))" and "EN 18031 test
> alignment").

The Radio Equipment Directive Article 3.3(d)(e)(f) (Delegated Regulation (EU)
2022/30) is mandatory for radio equipment from 1 August 2025; EN 18031-1/2/3 are
the harmonised standards. This guide will map network protection, personal-data
protection, and fraud prevention onto what the stack provides.
